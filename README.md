# roll-buddy

一个可视化的 Claude Code Buddy 生成器。

## 功能

- **物种选择**：支持 18 种物种（duck、goose、blob、cat、dragon、octopus 等）
- **稀有度筛选**：common、uncommon、rare、epic、legendary 五种稀有度
- **外观定制**：可选眼睛样式、帽子装饰
- **闪光判定**：支持筛选闪光 (shiny) Buddy
- **双模式生成**：
  - 随机模式：生成随机用户 ID 进行搜索
  - 顺序模式：按前缀 + 索引生成 ID
- **哈希算法**：支持 Bun/Wyhash 和 Legacy/FNV-1a 两种算法
- **自定义参数**：可配置随机字节长度、最大尝试次数、返回数量

## 使用说明

1. 在浏览器中打开 `buddy.html`
2. 配置筛选条件（至少选择一项）：物种、稀有度、眼睛、帽子、闪光
3. 配置运行参数：
   - 选择哈希算法：原生版选择 Bun；npm 使用 Legacy
   - 设置字节长度、尝试次数、返回数量
4. 点击「开始搜索」按钮
5. 在结果区域查看匹配的用户 ID，点击「复制」按钮。修改 `~/.claude.json` 文件：
   - 替换 `userID` 值
   - 删除 `companion` 对象。

```json
{
    ...
    "userID": "生成的用户 ID",
    
    // 删除 companion
    "companion": {
      "name": "Moth",
      "personality": "A legendary chonk of few words.",
      "hatchedAt": 1775099398412
    }
    ...
}
```

## Buddy 介绍

### 启用步骤

#### 版本要求

| 前置条件             | 要求                   | 说明                         |
| -------------------- | ---------------------- | ---------------------------- |
| **Claude Code 版本** | 2.1.89 或更高          | 使用 `claude --version` 查看 |
| **订阅计划**         | Claude Code Pro 及以上 | 免费用户暂不可用             |

#### 启用命令

```bash
# 1. 检查并升级 Claude Code 版本
claude --version
npm install -g @anthropic-ai/claude-code@latest

# 2. 在 Claude Code 终端中输入
/buddy

# 3. 与 Buddy 互动
/buddy pet    # 抚摸宠物
/buddy off    # 隐藏 Buddy
```

### 属性系统

每只 Buddy 拥有 5 个属性值，决定它的性格和互动风格：

| 属性          | 说明     | 高值表现           |
| ------------- | -------- | ------------------ |
| **DEBUGGING** | 调试能力 | 更擅长指出代码问题 |
| **PATIENCE**  | 耐心程度 | 更温和的评论风格   |
| **CHAOS**     | 混乱程度 | 更多不可预测的吐槽 |
| **WISDOM**    | 智慧水平 | 更深刻的技术洞察   |
| **SNARK**     | 毒舌程度 | 更犀利的点评风格   |

每只 Buddy 有一个**峰值属性**（最高值）和一个**低谷属性**（最低值），其余随机分布。稀有度越高，总属性值越高。

### 互动玩法

#### 基础命令

```bash
/buddy      # 显示 Buddy（首次触发孵化）
/buddy pet  # 抚摸宠物（爱心效果）
/buddy off  # 关闭显示
```

#### 互动方式

- **被动观察**：Buddy 自动观察对话，在关键时刻通过气泡发表评论
- **主动对话**：在输入中提到 Buddy 的名字，它会回应你
- **性格驱动**：回应风格由属性决定（高 SNARK 犀利吐槽，高 PATIENCE 温柔安慰）
